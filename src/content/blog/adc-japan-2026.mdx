Earlier today I gave a talk at [ADC Japan](https://adcjapan.com/) called **Making Beats with React Native**. The recording isn't available yet, but I wanted to put all the links in one place for anyone who attended — or anyone curious about cross-platform audio.

If you were there: thank you for coming. If you weren't: here's what you missed and where to learn more.

## The short version

I build an app called [Midicircuit](https://apps.apple.com/gb/app/midicircuit/id1558844679) — a music-learning app where people can play with beats, instruments, and interactive lessons. The hardest part was never the UI. It was the audio layer underneath everything.

The talk walked through how [Elementary Audio](https://www.elementary.audio/) — a functional approach where you describe audio graphs in JavaScript and let native runtimes render them as real-time sound — plus React Native made cross-platform mobile audio practical. MidiCircuit for Android is now in beta as a direct result of that shift.

## Links from the talk

### Open source

- **[elementary-audio-kit](https://github.com/txbrown/elementary-audio-kit)** — A React component library for musical ideas. Instruments, sequencers, transports, drum pads — reusable building blocks on top of Elementary Audio. Available on [npm](https://www.npmjs.com/package/elementary-audio-kit).

- **[obsidian-elementary-audio](https://github.com/yonkolevel/obsidian-elementary-audio)** — An Obsidian plugin that turns markdown into playable instruments. Write a fenced code block, get a drum machine or piano. I [wrote about it here](/blog/obsidian-elementary-audio) if you want the deeper story.

## The thesis

> Elementary Audio + React Native makes expressive cross-platform mobile audio practical today.

Not magic. Not "all audio problems are solved." But the center of gravity changes: the musical idea is no longer trapped in one platform's audio engine. You describe an audio graph in JavaScript. A web or native runtime renders it as real-time sound. React Native gives you a practical path to bring that graph to mobile.
